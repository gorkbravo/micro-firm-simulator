# Micro Firm Simulator

Interactive teaching activity for Microeconomics I, built around Varian-style firm theory topics and Dominican company cases.

## Contents

- `index.html`: standalone simulator. Open it in a browser to run the classroom activity.
- `micro-firm-simulator.html`: same simulator with the original working filename.
- `docs/guion-operativo.md`: classroom flow and checklist.
- `materials/`: source lecture presentations used to align the rounds.

## Classroom Flow

The simulator rounds follow the chronological order of the course:

1. Unidad XI: tecnologia.
2. Unidad XII: maximizacion del beneficio.
3. Unidad XIII: minimizacion de costes.
4. Unidad XIII: precios relativos and conditional factor demands.
5. Unidad XIV: curvas de costes.
6. Reveal/final formalization.

## Notes

The Dominican companies and production functions are didactic simulations, not empirical estimates of those firms.
